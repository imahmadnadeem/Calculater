# LED Display Calculator

A professional web application designed to calculate LED display resolutions for Adobe After Effects projects. This tool streamlines video production workflows by providing accurate pixel dimensions based on physical LED display specifications.

## Features

* Calculate total pixel dimensions based on physical LED display specifications.
* Real-time validation and error handling.
* Responsive design that works flawlessly on mobile, tablet, and desktop.
* Clear results display formatted specifically for After Effects compositions.
